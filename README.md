# Personal Website

Welcome to my corner of the internet - a place for my projects, thoughts,
and things I'm working on.

## Technologies Used

- **JavaScript (ES6+)**: Interactivity
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Astro**: Layout and component based static site builder
